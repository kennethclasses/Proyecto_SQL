# Proyecto_SQL
Analisis de viajes de taxis por aplicación

Simulamos trabajar como analista para Zuber, una nueva empresa de viajes compartidos que se está lanzando en Chicago. 
La tarea es encontrar patrones en la información disponible en cuanto a viajes y datos del clima para comprender las preferencias de los pasajeros y el impacto de los factores externos en los viajes.

Creado por Kenneth Toral en VS Code usando Python.

Para una lista de los requerimentos ver requirements.txt
